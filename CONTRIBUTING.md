# Contributing

This repository is a [GitHub Skills](https://skills.github.com/) exercise for building applications with GitHub Copilot agent mode.

## How to use this exercise

1. Click the **Copy Exercise** button on the [exercise page](https://github.com/xabierlameiro/skills-build-applications-w-copilot-agent-mode)
2. Follow the step-by-step instructions in the Issues tab of your copy
3. Each step builds on the previous one to create a full-stack fitness application

## Additional resources

- [GitHub Copilot Documentation](https://docs.github.com/en/copilot)
- [GitHub Copilot in VS Code](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot)
- [GitHub Skills](https://skills.github.com/)

## Reporting issues

If you find a problem with the exercise itself (not your implementation), open an issue in this repository.
